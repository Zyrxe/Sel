PROJECT NAME: ALONEA (ALO) – Sustainable Web3 Ecosystem on BSC

SMART CONTRACT REQUIREMENTS (Solidity ^0.8.27)

1. ALOToken.sol (BEP-20)

· Token Details: Name: "ALONEA", Symbol: "ALO"
· Total Supply: 100,000,000 ALO
· Transaction Fees: 2% total
  · 1% Buyback wallet
  · 0.5% Auto-add liquidity
  · 0.5% Treasury wallet
· Features: Auto buyback & burn, deflationary supply mechanism
· Security: ReentrancyGuard + Ownable from OpenZeppelin

2. ALOStaking.sol (Tier-based Staking)

Staking Tiers:

· Free: 0 ALO (1.0x multiplier)
· Bronze: 100 ALO (1.0x multiplier)
· Silver: 500 ALO (1.1x multiplier)
· Platinum: 1,000 ALO (1.25x multiplier)
· Diamond: 1,500 ALO (1.5x multiplier)

Functions: Stake, Unstake, Claim Rewards, Emergency Withdraw
Features: Live reward calculation based on time and multiplier

3. ALOGovernance.sol

· Proposal creation and management
· Voting power based on staked ALO balance
· Quorum requirements and execution logic
· Time-locked proposals

4. ALOBuyback.sol

· Automatic BNB to ALO conversion
· Buyback mechanism with burn function
· Treasury-controlled parameters

All contracts must:

· Compile under Solidity ^0.8.27
· Use latest OpenZeppelin contracts
· Include comprehensive event logging
· Have reentrancy protection where needed

DEVELOPMENT WORKFLOW

Prerequisites:

· Node.js 18+
· npm or yarn
· Git
· MetaMask or Web3 wallet

Setup Commands:

```bash
# Install dependencies
npm install

# Compile contracts
npx hardhat compile

# Run tests
npx hardhat test

# Deploy to testnet
npm run deploy:testnet

# Start frontend
cd frontend && npm run dev

# Build for production
npm run build
```

Environment Files:

.env.example:

```
BSC_MAINNET_RPC=https://bsc-dataseed.binance.org/
BSC_TESTNET_RPC=https://data-seed-prebsc-1-s1.binance.org:8545/
WALLETCONNECT_PROJECT_ID=your_project_id
DEPLOYER_PRIVATE_KEY=your_private_key
```
